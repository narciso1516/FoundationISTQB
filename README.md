# ISTQB CTFL — Guía educativa

Guía web interactiva para estudiar los fundamentos de testing incluidos en ISTQB Certified Tester Foundation Level.

## Objetivo

Presentar los conceptos de testing de forma clara, visual y navegable, con capítulos y recursos orientados al aprendizaje.

## Tecnología

- HTML
- CSS
- JavaScript
- Diseño responsive
- Componentes de experiencia educativa

## Uso

Abre `index.html` en un navegador.

## Aviso

Este es un recurso personal de estudio. ISTQB y sus marcas pertenecen a sus respectivos titulares. El repositorio no representa una publicación oficial de ISTQB.
